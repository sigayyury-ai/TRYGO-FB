## 🔄 Процесс работы

### Шаг 1: Выделение текста
1. Пользователь выделяет часть текста в редакторе (стандартное выделение)
2. Появляется визуальная панель с выделенным текстом

### Шаг 2: Визуальная панель выделения

```
┌────────────────────────────────────────────┐
│ ✨ Selected text (45 chars)                │
│                                             │
│ "Second, create a minimum viable product   │
│  to test your assumptions..."              │
│                                             │
│  [+] [X]                                    │
└────────────────────────────────────────────┘
```

**Элементы:**
- Иконка ✨ (Sparkles) - показывает, что это AI-функция
- Показывает первые 200 символов выделенного текста
- Кнопка **[+]** - добавить в чат для редактирования
- Кнопка **[X]** - очистить выделение

### Шаг 3: Добавление в чат
При нажатии на **[+]**:
- Выделенный текст появляется в чате как контекст
- В поле ввода показывается подсказка: "How should I edit this text?"
- Появляется индикатор "Editing: [текст]"

### Шаг 4: Редактирование через чат

**В чате:**
```
┌─────────────────────────────────────────────┐
│ 📝 Selected text for editing:               │
│                                              │
│ "Second, create a minimum viable product    │
│  to test your assumptions..."                │
│                                              │
│ How would you like me to edit this?         │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ User: make it more engaging                 │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ ✅ Selected text rewritten:                 │
│                                              │
│ "Second, develop a simple prototype that    │
│  validates your core assumptions and helps   │
│  you gather real user feedback quickly..."   │
└─────────────────────────────────────────────┘
```

### Шаг 5: Применение изменений
- **Только выделенная часть** заменяется в редакторе
- Остальной текст остается без изменений
- Выделение очищается автоматически
- В чате показывается подтверждение

## 🎯 Ключевые особенности

### ✅ Частичное редактирование
- Переписывается **только** выделенная часть
- Контекст до и после сохраняется
- Остальной контент не затрагивается

### ✅ Визуальная обратная связь
- Четкое выделение выбранного текста
- Индикатор активного редактирования в чате
- Показ результата перед применением

### ✅ Интуитивный workflow
1. Выделить → 2. Нажать [+] → 3. Написать инструкцию → 4. Получить результат → 5. Применить

## 📱 Адаптивность

На мобильных устройствах:
- Две колонки могут переключаться в вертикальный layout
- Чат может быть внизу или открываться как модальное окно
- Панель выделения адаптируется под размер экрана

## 🎨 Цветовая схема

- **Панель выделения**: Фиолетовая (`purple-50` фон, `purple-300` рамка)
- **Кнопка [+]** : Фиолетовая (`purple-100` фон)
- **Чат**: Синий для пользователя, белый для AI
- **Индикатор редактирования**: Фиолетовый фон

## ✅ Преимущества

1. **Точное редактирование** - меняется только нужная часть
2. **Визуальная ясность** - видно, что именно редактируется
3. **Интерактивность** - диалог с AI через чат
4. **Безопасность** - остальной контент не затрагивается
5. **Гибкость** - можно редактировать несколько частей по очереди
